# Art-Daily
这是我模仿知乎日报做的一个hybird应用，cordova打包，ionic框架

#feature:
>###Client:
- Andriod App
- side bar
- Sliding effect
- fastclick
- login
- select art category

>###Server:
- Nodejs + Express
- Login
- Signin
- admin:Editor(edit daily article)
- admin:Panel(article manager)
